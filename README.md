# 🎮 Simple Memory Game Project

This is a Simple TypeScript-based game built with ES6 features, featuring stunning visuals, immersive audio, and addictive gameplay. Dive into a world of fun and challenge!

## 📋 Features

- **Dynamic Gameplay**: Experience fast-paced action with multiple levels and challenges.
- **Audio Integration**: Enjoy immersive sound effects and background music.
- **Responsive Design**: Optimized for desktop and mobile devices.
- **TypeScript Powered**: Built with modern TypeScript for better code quality and maintainability.
- **Custom Assets**: Includes custom images and audio files for a unique experience.

## 🛠️ Technologies Used

- **TypeScript**: For type-safe JavaScript development.
- **HTML5 & CSS3**: For structure and styling.
- **ES6 Modules**: For modular code organization.
- **Audio API**: For sound management.

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/awesome-game.git
   cd awesome-game
   ```
2. Install dependencies:

   ```bash
   npm install
   ```
3. Build the project:

   ```bash
   npm run build
   ```
4. Open `index.html` in your browser to play!

### Running Locally

- Use a local server to avoid CORS issues:
  ```bash
  npx http-server
  ```
- Or open `index.html` directly in a modern browser.

## 🎯 How to Play

- Use arrow keys or WASD to move.
- Collect items and avoid obstacles.
- Reach the highest score possible!
